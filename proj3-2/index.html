Part 1. Mirror and Glass Materials

Show a sequence of six images of scene `CBspheres.dae` rendered with `max_ray_depth` set to 0, 1, 2, 3, 4, 5, and 100. The other settings should be at least 64 samples per pixel and 4 samples per light. Make sure to include all screenshots.


Point out the new multibounce effects that appear in each image.

In this scene, we see that there is a sphere that acts like a mirror (relfects light) and another sphere that primarily refracts light. When the max ray depth is 0, as expected the only illumination we get is internal, so only the light source is seen. When it is 1, the reflective sphere only reflects the direct lighting, while the refractive sphere refracts only the small patch where the direct light hits. At 2 we start to see some more interesting features. The mirror sphere gets the rays that bounce off the walls and the other sphere, while the "glass" sphere is starting to show some more refracted light. As we increase the ray depth to 3, 4, 5, and 100, the images reflected/refracted in the spheres become increasingly clear and the scene becomes brighter (note that the difference as we increase the ray depth becomes less noticable since the likelihood that a ray would bounce that much decreases significantly).


Explain how these bounce numbers relate to the particular effects that appear. Make sure to include all screenshots.

We can see that for 0 bounce, we don't see any reflection or refraction as only internal lighting is seen. At 1 bounce, the only reflection is at the very top of the left sphere as that is what recieves direct lighting from the light source, and we only see a small patch of refraction at the top of the right sphere from the direct lighting. At 2 bounces we see more effects such as the light from the walls and the other spheres interacting with each other, since to see reflection from other objects we need a minimum of 2 bounces (light source -> object -> reflective/refractive surface). As we increase the number of bounces, we see more and more detail appear in the spheres.


Part 2. Microfacet Material

Show a screenshot sequence of 4 images of scene `CBdragon_microfacet_au.dae` rendered with $\alpha$ set to 0.005, 0.05, 0.25 and 0.5. The other settings should be at least 128 samples per pixel and 1 samples per light. The number of bounces should be at least 5. Describe the differences between different images. Note that, to change the $\alpha$, just open the .dae file and search for `microfacet`.

We can see through the 4 images that as we increase alpha (a parameter which represents the roughness of the material), the image becomes less glossy and more matte. We can also see that there is less specularity as alpha is increased.


Show two images of scene `CBbunny_microfacet_cu.dae` rendered using cosine hemisphere sampling (default) and your importance sampling. The sampling rate should be fixed at 64 samples per pixel and 1 samples per light. The number of bounces should be at least 5. Briefly discuss their difference.

We can see that using cosine hemisphere sampling, the scene is much more noisy (for a fixed number of samples per pixel). This is expected since the samples are not drawn from the light source as they are in importance sampling. To increase the quality, we would have to increase the number of samples with cosine hemisphere, but as we can see we don't need to do so with importance sampling to achieve the same result.
